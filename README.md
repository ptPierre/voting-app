# Voting

Overview
A brief description of the project.

## Requirements
Node.js: Make sure you have Node.js installed. You can download it from Node.js.
npm: npm (Node Package Manager) is installed with Node.js.

## Installation
Follow these steps to clone the repository and install the necessary packages:

Clone the repository
Open your terminal and run:

```bash
git clone https://github.com/ptPierre/voting-app.git
```

Navigate into the project folder

```bash
cd voting-app
```

## Install dependencies

Run the following command to install all necessary packages:

```bash
npm install
```
## Running the Project

To start the project, run:

```bash
npm start
```
This command will start the development server 

## Additional Scripts
Build: To build the project for production, use:

```bash
npm run build
````

Test: To run tests, use:

```bash
npm test
```
